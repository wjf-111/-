# 活动海报插入说明

## 📋 需要做的事情

您的票务系统已经支持显示活动海报，但需要您准备海报图片文件。以下是具体步骤：

## 📁 第一步：创建图片文件夹

在 `数据库` 文件夹下创建一个名为 `图片` 的文件夹（如果还没有的话）。

## 🖼️ 第二步：准备海报图片

根据 `data.js` 中的配置，您需要为以下 12 个活动准备对应的海报图片：

### 需要准备的海报文件列表：

1. **周杰伦 2025 世界巡回演唱会 - 北京鸟巢站.jpg**
2. **《如梦之梦》话剧巡演 - 上海大剧院站.jpg**
3. **2025 中超联赛北京国安 VS 广州恒大.jpg**
4. **「笑果脱口秀」年度巡演 - 深圳站.jpg**
5. **「梵高星空」沉浸式艺术展览 - 成都站.jpg**
6. **2025 草莓音乐节 - 杭州站.jpg**
7. **《冰雪奇缘》儿童剧 - 广州少年宫站.jpg**
8. **2025 黄山风景区深度游套票 - 暑期专场.jpg**
9. **2025 商业领袖峰会 - 上海国际会议中心站.jpg**
10. **「漫威宇宙」电影展映周 - 北京万达影城.jpg**
11. **f3cffd44b03464c560b2a534bb6999d7.jpg** （2025 乒乓球世界杯决赛的默认图片）
12. **「李健」民谣演唱会 - 南京奥体中心站.jpg**

## 📂 文件存放位置

将所有海报图片文件放入：
```
数据库/图片/
```

## ✅ 图片要求

- **格式**：建议使用 JPG 或 PNG 格式
- **文件名**：必须与上面列表中的文件名**完全一致**（包括中文字符和标点符号）
- **大小**：建议图片宽度在 800-1200 像素之间，高度根据比例调整
- **默认图片**：如果某个活动的海报缺失，系统会自动使用 `f3cffd44b03464c560b2a534bb6999d7.jpg` 作为默认图片

## 🔍 如何检查

1. 确保 `数据库/图片/` 文件夹存在
2. 将海报图片放入该文件夹
3. 打开 `index.html` 查看活动列表，海报应该会自动显示
4. 点击活动进入详情页，详情页顶部也会显示海报

## 📝 注意事项

- 文件名必须与 `data.js` 中 `activityImageMap` 映射的名称完全匹配
- 如果图片加载失败，会自动使用默认图片 `f3cffd44b03464c560b2a534bb6999d7.jpg`
- 建议准备一张默认海报图片，以防某些活动海报缺失

## 🎨 海报显示位置

海报会在以下位置显示：
1. **活动列表页**：每个活动卡片左侧显示海报缩略图
2. **活动详情页**：详情页顶部显示完整海报

---

**提示**：如果您已经有海报图片但文件名不同，可以：
1. 重命名图片文件以匹配上述列表
2. 或者修改 `data.js` 中的 `activityImageMap` 映射来匹配您的文件名
